# Портфолио Richter

Минималистичное портфолио разработчика с элементами кодинга.

## Технологии

- **React 18** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **Vite** - быстрый инструмент сборки
- **Tailwind CSS** - utility-first CSS фреймворк
- **shadcn/ui** - набор переиспользуемых компонентов
- **Framer Motion** - библиотека анимаций
- **JetBrains Mono** - моноширинный шрифт для кода

## Установка

```bash
cd web
npm install
```

## Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу http://localhost:5173

## Сборка

```bash
npm run build
```

## Дизайн-система

Дизайн-система документация находится в файлах:
- `DESIGN_SYSTEM.md` - полное описание дизайн-системы
- `design-system.json` - JSON конфигурация дизайн-системы
- `.cursor/rules/design-system.mdc` - правила для Cursor AI

### Основные принципы

- **Минималистичный стиль** - плоские цвета, тонкие границы, без теней
- **Элементы кодинга** - терминал, code блоки, моноширинный шрифт
- **Темная тема** - черный фон (#0a0a0a), серые акценты
- **Моноширинный шрифт** - для навигации, кнопок, кода
- **Нижний регистр** - для навигации и меток

## Компоненты

### Секции
- **Hero** - главный экран с терминалом
- **About** - информация о разработчике с code блоком
- **TechStack** - технологический стек
- **Projects** - проекты с фильтрацией
- **Contact** - форма контакта и социальные сети

### UI Компоненты
- **Terminal** - анимированный терминал
- **CodeBlock** - блок кода с подсветкой
- **Card, Button, Badge** - базовые компоненты
- **Navigation, Footer, ScrollProgress** - layout компоненты

## Добавление компонентов shadcn/ui

```bash
npx shadcn-ui@latest add [component-name]
```

## Правила дизайна

См. `DESIGN_SYSTEM.md` для полной документации. Основные правила:
- НЕТ градиентов, теней, свечений
- ДА плоские цвета, тонкие границы, моноширинный шрифт
- Темные фоны, серые оттенки, минимальные скругления
